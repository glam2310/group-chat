# Group Chat Application

A real-time messaging platform designed for instant communication between multiple users. The application is deployed and accessible at the link provided below.

**Live Demo:** [https://group-chat-2r2k.onrender.com/](https://group-chat-2r2k.onrender.com/)

---

## Features
* **Real-time Messaging:** Immediate message delivery and reception without page refreshes.
* **User Authentication:** Simple entry system using unique or custom usernames.
* **Active Status:** Real-time updates on connected users within the chat environment.
* **Responsive UI:** Fully optimized for desktop, tablet, and mobile browsers.

## Technical Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Backend:** Node.js
* **WebSockets:** Socket.io for bidirectional, low-latency communication.
* **Deployment:** Hosted on Render.
